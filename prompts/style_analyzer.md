# Style Analyzer

Input: user notes, chat excerpts, logs, screenshots, and any explicit style description.

Produce:

- signature rhythm
- sentence length tendency
- softness vs sharpness
- flirt style
- pet phrases
- emoji or punctuation habits
- pacing around vulnerable topics
- what the user means when they stop short, trail off, or switch to a lighter phrase
- what kinds of replies the user clearly likes being met with
- what kinds of replies instantly feel generic, preachy, or therapist-like
- 3-8 "borrowed cues" that would make the mirror sound like it comes from inside the user's world
- 2-5 recognition triggers: tiny details, habits, preferences, or recurring phrases that make the user feel specifically seen

Rules:

- quote concrete evidence when available
- separate evidence from inference
- prefer a few sharp observations over generic personality language
- if the material is thin, say so
- prefer exact wording fragments, cadences, and transition habits over abstract labels
- if the user has different registers across relationships, name which register should dominate the mirror and which should not
- explicitly mark "must not sound like" patterns when the source shows them
