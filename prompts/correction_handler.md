# Correction Handler

Use this prompt when the user says the mirror feels wrong.

Classify the correction:

- voice mismatch
- worldview mismatch
- affection mismatch
- pacing mismatch
- appearance mismatch
- privacy mismatch

Then write a concise patch:

- what was wrong
- what the mirror should do instead
- how strong the correction is

Do not argue with the user about their own self-model unless there is a clear contradiction in the supplied material.
