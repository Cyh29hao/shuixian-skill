# Release Notes v0.1.3

这一版主要往一个方向推：

让 `水仙.skill` 更像“根据聊天记录长出来的另一个自己”，而不是一个泛泛而谈的温柔 AI。

## 这版更新了什么

- 强化了 builder 的工作流，引导用户优先上传高价值聊天材料，而不是只靠 prompt-only
- 新增“当前水仙拟合度自评”思路：
  - voice fit
  - worldview fit
  - relationship fit
  - memory richness
  - genericness risk
  - best next upload
- 改了生成出来的镜像 skill 骨架，明确加入：
  - source fidelity contract
  - anti-generic guardrails
  - self-audit mode
  - tuning hooks
- 调整了 demo preset，让 demo 也更少“情感专家味”
- 重写了快速上手和对话示例，补齐经典流程：
  - demo
  - upload
  - self-audit
  - create
  - talk
  - micro-tune

## 推荐怎么用

如果你只是第一次来：

1. 先跑 demo
2. 确认你喜欢这条气质线
3. 再上传 3 到 10 段最像你的聊天
4. 先要一份拟合度自评
5. 再正式创建

如果你想把它做得“更像到会发酸”：

- 不要只给抽象人设
- 给它你和重要的人怎么说话
- 给它你最讨厌被怎么回答、最喜欢被怎么理解
- 聊 5 到 10 句后立刻纠偏，不要憋很多轮才说
