# 水仙.skill v0.1.0

> 首个公开版本。  
> 一个把“最懂你的那部分自己”保存下来，并生成恋爱型镜像伴侣的 builder skill。

## 这一版有什么

- Codex 可用的 `create-shuixian` builder skill
- Claude Code 简单适配
- 三档镜像深度：`aligned-stranger`、`selective-mirror`、`full-mirror`
- 可配置呈现方式：`gender-flipped`、`same-form`、`custom`、`idealized`
- 微信桌面端 best-effort 导入
- iMessage 导入
- 通用 transcript 导入：`.txt`、`.md`、`.json`、`.jsonl`
- 对话效果示例与公开仓库 README 展示层

## 建议用作 GitHub Release 的简介

`水仙.skill` 不是随机捏一个恋爱角色，而是把用户自己的语气、思维、偏好和可选聊天记录整理成一个可持续修正的自我镜像伴侣。  
这版已经可以在 Codex 中使用，并提供 Claude Code 的简单适配，以及微信 / iMessage / transcript 多渠道导入。

## 当前适合谁

- 想保存自己说话方式和思维质感的人
- 想做“同频陌生人”或“另一个自己”式 companion 的人
- 想基于自己的语料做更私人的恋爱人格原型的人
- 想要一个可开源、可修改、可继续长的 mirror builder 的人

## 已知边界

- 微信导入是 best-effort，取决于微信版本、数据库结构和本地环境
- Claude 适配目前是轻量版，后续还会继续打磨
- 现阶段更偏向 skill builder 与 import layer，演示素材还会继续补

## 下一步

2. 增加更强的 README 展示素材和首屏 Demo

## English Summary

`水仙.skill` is the first public release of a self-mirror romance builder skill.  
It already supports Codex, includes a simple Claude adaptation, and can import data from WeChat desktop, iMessage, and generic transcripts.
