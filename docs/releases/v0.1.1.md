# 水仙.skill v0.1.1

> 可用性和展示层更新。  
> 这一版重点不是“再多加一点功能”，而是让第一次来到仓库的人更容易真的跑起来。

## 这一版新增

- `tools/demo_builder.py`
  不需要任何私密数据，也能一键生成公开 demo mirror。
- `docs/quickstart.md`
  把试玩、初版自定义、聊天记录导入拆成了三条最直接的路径。
- `examples/starter-pack/`
  给手动创建 mirror 的用户一个现成的起步模板。
- `assets/hero-preview-v7.png` 与 `assets/quickstart-flow-v8.png`
  README 使用更稳定的 PNG 首图与 quick start 流程图。
- README / README_EN 更新
  更强调“先试再导入隐私数据”的使用顺序。

## 为什么做这版

如果一个 skill 只有“功能很多”，但第一次使用时不知道从哪里下手，它就很难变成真正会被 star、被推荐、被复用的项目。

所以这版主要在做两件事：

1. 让别人不用先交出聊天记录，也能感受到项目的效果。
2. 让第一次使用的人能在几分钟内完成从试玩到自定义的第一步。

## English Summary

`v0.1.1` focuses on first-run usability.

It adds:

- a public demo builder with no private data required
- a dedicated quick start guide
- starter files for manual mirror creation
- visual README preview assets
